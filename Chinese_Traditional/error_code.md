# 錯誤碼

## HTTP請求錯誤碼

| 錯誤碼 | 含義 | 說明 |
| --- | --- | --- |
| 200 | 成功 |  |
| 403 | 資源不可用 | header頭資訊錯誤或不全 |
| 404 | 資源未找到 | url錯誤 |
| 405 | 請求方式錯誤 | GET、POST方式替換 |
| 500 | 伺服器錯誤 | 需聯繫平臺客服 |

## 介面返回數據code碼

| 錯誤碼 | 含義 | 說明 |
| --- | --- | --- |
| 0 | 成功 |  |
| 1000 | 請求缺少參數或參數格式錯誤 | 請對照文檔參數 |
| 1200 | 沒有找到合法的api_key | 請求參數中沒有api_key |
| 1201 | 認證失敗 | 簽名認證失敗 |
| 1206 | api介面缺少參數sign | 請求參數中沒有sign |
| 24001 | api數據沒有發現 | api_key錯誤 |
